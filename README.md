# 🐷 Help_Brickto
**게임잼 제시 키워드 (빨간색, 벽돌, 겨울) 중 빨간색과 벽돌 키워드를 선택하여 제작한 게임 프로젝트**

---

## 🎮 프로젝트 소개  
- **Brickto** 는 플레이어가 제한 시간 내에 맵을 탈출해야 하는 3인칭 퍼즐 게임입니다.
- 기름통을 넘어뜨려 쫓아오는 돼지를 따돌리고, 구조 요청을 외치는 NPC를 구출해야 하는 **짧고 몰입감 있는 게임플레이** 를 목표로 제작했습니다.

### 🔧 기능 요약
- 적 NPC가 플레이어 감지 시 추적 상태로 전환, 미감지 시 3개 베이스를 순회하며 탐색

- 기절 상태 구현: Oil 장애물에 닿으면 기절, 일정 시간 후 랜덤 베이스로 복귀

- 플레이어 감지 범위와 상태를 시각적으로 표현하는 레이더 구현 (원형 링 + 부채꼴 스캔 효과)

- 레이더 색상(녹색/빨간색), 투명도, 회전 애니메이션으로 탐지 상태를 직관적으로 표시
